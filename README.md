# Retail Store Bank Statement Analysis

## Project Objective
To analyze a retail store bank statement by categorizing transactions into accounting heads and generating a daily financial summary.

## Process
- Converted debit and credit into a single amount column.
- Categorized transactions into Sales, Purchases, Bank Charges, Utilities, and Others.
- Created pivot tables to summarize daily activity and category totals.

## Key Insights
- Sales account for the majority of inflows.
- Purchases represent the largest outflow.
- Bank charges occur consistently.
- Some transactions required manual review for proper classification.

## Tools Used
- Microsoft Excel (Data cleaning, Pivot Tables, Financial summary)

## Outcome
A structured daily financial report showing cash inflows, outflows, and net position.
